布兰登·桑德森的入坑建议

2020-02-02

从哪本书入坑

  • 如果你不经常看奇幻
    试试《审判者传奇：钢铁心》或《伊岚翠》。

  • 如果你是奇幻爱好者
    试试《迷雾之子：最后帝国》或《飓光志：王者之路》。

  • 如果你想看言情作品
    试试《破战者》。

  • 如果你想看搞笑作品
    试试《阿尔卡特拉兹与黑暗眼镜侠》。

三界宙作品的推荐阅读顺序

推荐按照作品分卷顺序阅读。建议选择以下作品入坑：

  • 《迷雾之子：最后帝国》
  • 《皇帝魂》
  • 《飓光志：王者之路》
  • 《伊岚翠》
  • 《破战者》

上述作品均含有别的三界宙作品的线索，但你的阅读体验不会受到影响。寻找彩蛋是重读小说时的乐趣之一。

《迷雾之子：执法镕金》也能作为起始点，但书中含有前三卷的剧透。

非三界宙作品的推荐阅读顺序

这些作品互不关联，推荐按照分卷顺序阅读。
